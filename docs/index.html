Web Widget

开放的 Web 前端的应用容器

入门指南 浏览文档

为什么需要 Web Widget？

随时集成

只需要知道应用的 URL 即可立即使用，就像将图片插入到网页中一样，无需编写胶水代码、亦无须测试。

安全运行第三方代码

它使用了 Web Sandbox BOM 虚拟化技术，它隔离了第三方代码潜在的 XSS 攻击、钓鱼链接等。

构建无代码编程

所有的应用都通过容器运行，而容器就是一个 HTML 标签，它容易被可视化工具排版、被服务端模板输出。

延长软件生命

它完全基于 Web 标准，不依赖任何的框架，但却可以和现在与未来的技术栈兼容，包括 React 与 Vue 等。

构建开放的插件系统

应用通过 NPM 托管，通过公共 CDN 可以直接使用它，并且具备在浏览器中自动更新版本的能力。

提高首屏速度

应用拥有 AMP 高性能组件系统的关键要素，有利于实现首屏性能领先的网页架构。
